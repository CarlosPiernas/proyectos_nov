Descripción de los archivos en esta carpeta.

	*16740: Carpeta con los datos de peluqueria

	1. Archivos SQL: En esta carpeta están los archivos en .sql de lo documentado en la documentacion
		a. Consultas.sql: son las consultas reglamentarias para la práctica
		b. index.sql: los diferentes índices creados.
		c. transaccion.sql: la transaccion completa.

	2. Datos para el generador: Son los archivos necesarios para que el generador de datos funcione correctamente

	3. Extras: Son los archivos SQL que corresponden a los extras de la práctica.
		a. datos_csv: los datos exportados con el archivo copia_a_csv.sql
		b. ConsultasExtra.sql: las consultas extra que se utilizan en la interfaz con las vistas dinámicas.
		c. copia_a_csv.sql: el código que genera la copia a csv.
		d. roles.sql: el código que crea los roles y sus diferentes permisos.

	4. Documentación Acceso a Datos.pdf: La documentación completa en archivo pdf

	5. GeneradorDatos.zip: El programa que genera los datos para la base de datos.

	6. Peluqueria.sql: El archivo sql con la base de datos peluqueria completa

	7. peluqueria_final_fixed.dbm: El archivo que genera el diagrama E-R en pgModeler

	8. README.txt: el archivo que estás leyendo ahora mismo.
